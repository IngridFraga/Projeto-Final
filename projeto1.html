Como usar os comandos simples de Web para desenvolver um Box Model

O presente site foi criado com o intuito de ensinar os alunos de Web 1 a desenvolverem um box model usando comando simples da programação. Este terá que ser o resultado obtido ao final do projeto:

OBS: vocês devem modificar seus modelos para que fiquem diferente do apresentado em aula.

De um título para a página em "title - title". E dentro do corpo da nossa pagina crie três div cada uma com uma classe caixa1, caixa2 e caixa3.Dentro de cada div digite um coloque um tag colocando um conteúdo em cada, como caixa1, caixa2 e caixa3. Em seguida, abra no navegador e veja o resultado. Por enquanto são 3 paragrafos. Agora vamos criar o novo arquivo css. Para isso repita os dois primeiros passos e de o nome do arquivo como estilos.css ou outro nome. Mas para nossas modificações surtirem efeito no site precisamos criar um link do css no html. crie a tag link no head do arquivo html e coloque no href=“estilos.css”. (...)
